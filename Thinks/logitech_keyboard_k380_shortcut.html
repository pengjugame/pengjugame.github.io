# 罗技K380键盘iPad快捷键总结

# 功能键区

esc 可以直接取消正在进行的输入不用一个个回删

F4和F7功能一样 单击：回到主桌面 双击：多任务管理，双击F7多任务管理 长按：唤醒siri

F6打开虚拟键盘 可以在虚拟键盘上打字，可能输入一些键盘上没有的符号，一些表情什么的。

F8F9F10表示音乐的上一首，暂停，下一首 F11-F12-ins 表示静音减少音量和增加音量

del 删除 back退格 tab正常tab键 caps lock可以用来切换中英文， 可以在设置中处理设置成切换大小写，也可以用ctrl+space进行中英文切换

# 组合功能

fn+tab截屏 fn+L锁屏，和windows一样

cmd+C和Cmd+V 复制粘贴

Fn+左右 选中以每次单个词的速度选中之前和之后的文字，和vim里的w一样 如果是向前选择，则碰到空格或者换行停止 如果是向后选择，则可以一直选择

Fn+上下 回到行首和行尾 fn+s 搜索 等于cmd+space

cmd+tab 可以快速切换应用 cmd+shift+tab向左切换app cmd+space 搜索

cmd+左右 等于 fn+上下 回到行的最左和最右 cmd+上下 最开头和最结尾 长按cmd可以打开菜单，用不同的软件时可以使用不同的快捷键盘cmd+shift+3：截图 cmd+shift+4：截图截完图，进入截图编辑状态

start+左右，可以以单词或者词语的形式移动，这个是移动，而fn+左右则是选择 start+上下，到行首和行尾巴

alt option+cmd+D　显示程序坞

# 其它文本编辑

cmd+back 删除当前光标之前到行首
cmd+c　　　　　　　复制
cmd+v　　　　　　　粘贴
cmd+a　　　　　　　全选
cmd+x　　　　　　　剪贴
cmd+i　　　　　　　 斜体
cmd+b　　　　　　　加粗
cmd+u　　　　　　 下划线
